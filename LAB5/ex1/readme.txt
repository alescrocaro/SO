Como compilar:
    make

Como executar:
    P1:
        ./P1 <operador1><operando><operador2>

    P2: 
        ./P2

Bibliotecas usadas:
    todas bibliotecas adicionais são usadas para o FIFO (IPC).

Exemplo de uso:
    terminal1:  
        ./P2
    
    terminal2:
        ./P1 2*2+1-3  

OBSERVAÇÃO:
    O algoritmo suporta apenas resultados positivos menores que dez. Devido ao pouco tempo disponível dos alunos para implementação.
