Como compilar:
    make

Como executar:
    server:
        ./server

    jogador: 
        ./jogador

    mestre: 
        ./mestre    

Bibliotecas usadas:
    todas bibliotecas adicionais são usadas para o FIFO (IPC).

Exemplo de uso:
    ./server
    ./mestre (interação feita pelo terminal(scan))
    ./jogador (interação feita pelo terminal(scan))
